Website 3.0

The OutboundEngine website I inherited was built using wordpress. When I was hired my first project was to redesign / rebuild this website. Due to timeline constraints we created a new template for this site rather than building assets new site. That site has been through 2 more iterations since. The three main areas I am improving with this site are the customer journey, the technology, and the UI.

Technology

Because the current site is built with wordpress, it has many dependencies and limitations. So this site will be built using no CMS and as few libraries as possible to maximize speed and efficiency. This is critical for a lead generation site because it has a dramatic effect on conversion rates. I will also replace the php based frameword with a more modern, javascript-based stack.

Marketing Funnels

Currently there are very few types of funnels on the site, and the leads are handled very similarly. This site's new design will include several new funnels of different types, and the leads will be handled uniquely based on which funnnel they came from.

UI Design

The last piece of this project is the most important. Throughout the iterations of the last site I ran dozens of A/B tests and collected data. This redesign uses all of that data to inform the design and reduce friction for the user as much as possible.

UI Design
